## como usar

* instruções de download

```
1. git clone https://github.com/VTramon/Book-finder.git
```
* Instale as dependencias e inicie o projeto em modo de desenvolvimento
```
2. yarn 
3. yarn dev
```

#### OU

```
2. npm install 
3. npm run dev
```
